Exemple : Héritage

Python

class Animal:
    def parler(self):
        return "Je fais du bruit"

class Chien(Animal):  # Héritage simple
    def parler(self):
        return "Woof!"

class Chat(Animal):  # Héritage simple
    def parler(self):
        return "Meow"

Dans cet exemple, la classe `Chien` et la classe `Chat` héritent de la classe `Animal`. Chacune redéfinit la méthode `parler` pour fournir un comportement spécifique.

Java

class Animal {
    public String parler() {
        return "Je fais du bruit";
    }
}

class Chien extends Animal {  // Héritage simple
    @Override
    public String parler() {
        return "Woof!";
    }
}

class Chat extends Animal {  // Héritage simple
    @Override
    public String parler() {
        return "Meow";
    }
}

En Java, l'héritage est réalisé avec le mot-clé `extends`. Ici, `Chien` et `Chat` héritent de `Animal` et redéfinissent la méthode `parler` avec leur propre implémentation.
